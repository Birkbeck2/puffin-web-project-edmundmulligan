Web Witchcraft and Wizardry Color Palette

Home > Color Palette

This page displays all color variables defined in colours.css. The color system includes three style variants (normal, subdued, and vibrant) with both light and dark themes. All color combinations meet at least WCAG 2.2 AA contrast standards (4.5:1 ratio). Normal theme and popovers meet AAA standards (7:1 ratio). Each card shows the background color with its corresponding text color applied, along with both CSS variable names and sample text to preview readability.
