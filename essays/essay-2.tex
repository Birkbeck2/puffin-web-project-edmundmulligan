\subsection{History and Inspiration}
The Web Witchcraft and Wizardry project, originally  used by Kingston University Code Dojo, was a pdf file with a single html file and some inline CSS and JavaScript to get students started. This was then developed into a web site in github with eight lessons \citep{mulligan2024witchcraftandwizardry}, although no consideration was given to aesthetics or usability. The current project aims to redesign the website with a focus on accessibility and visual design, while maintaining the original Code Dojo concept of teaching kids to code by coding \citep{coderdojo2026}.

Inspiration for the colours of the new website were taken from the Embodied Mind logo, see figure \ref{fig:embodiedmind}. The vibrant style was drawn from 1960's art such as the work of Roy Lichtenstein (e.g. \cite{lichtenstein1963whaam}) and the television series Batman \citep{batman1966}. See figures \ref{fig:wham} and \ref{fig:batman} for examples of this style using bold colours and dramatic contrasts. The subdued style, for users who do not want the strident colours of the other two styles, uses a desaturated theme, arguably evoking a pre-colour black and white television look, although this is admittedly a tenuous link.
